# pathway-specific-targeting_Islets
Scripts corresponding to inter-organ genetic correlation analyses based on pathway-specific targeting

The R scripts provided in script for crosstissue correlation with pancreas pathways.R
Perform cross-organ correlations using a filtered version of GTEx (v8) to retain maximal representation across tissues. Studies developing and using these data can be found in: https://elifesciences.org/articles/76887 and https://elifesciences.org/articles/88863

These analyses focus on GO Terms corresponding to insulin secretion whcih can be easily exchanged for any pathway in any organ
